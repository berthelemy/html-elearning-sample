# Elearning sample requirements

## Summary

You are an expert elearning developer. I want you to build a SCORM 1.2 package designed to teach a student working at level CEFR A2 how to use the imperfect tense in French.

## Interactions

The package should contain examples of the following types of interaction:

- Click to reveal
- Flip the cards to reveal
- Match the words
- Multiple choice quiz

## Pages

The package should contain the following pages:

1. Introduction
2. When to use the imperfect tense
3. How to make the imperfect tense for er verbs
4. Quiz

## Scoring

The final score will be based on the number of correct answers given during the final quiz.

A Pass will be given if the user achieves more than 80%.

## Navigation

The user should be able to freely navigate to all the lessons within the package, or should be given the option to move onto the next or previous lesson.

## Accessibility

The package should meet the requirements of WCAG 2.0 Level AA.

## Graphic design

The package should follow best practices in visual hierarchy. The fonts and layouts used should meet the needs of a 30 year old person trying to learn French in order to move to France.

## Frameworks

Use the following frameworks to help build the package:

- SCORM: Pipwerks scorm API wrapper - https://github.com/pipwerks/scorm-api-wrapper
- Bootstrap 5
- Jquery (or another suitable Javascript library)
